# Car List Manager

## Features

- Parses car data from a local HTML file
- Displays vehicle name, price and mileage
- Calculates average vehicle price
- Calculates total vehicle value
- Exports parsed data to `cars.json`

## Technologies

- Python
- BeautifulSoup
- HTML
- CSS
- JSON
- Git / GitHub

## How to run

1. Install dependencies:

```bash
pip install beautifulsoup4
```

2. Run the app:

```bash
python main.py
```
